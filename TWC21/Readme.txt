 Run "Main.m" file to start the simulation.

Subfunctions:
1. "DS_AMP.m"  is the proposed Algorithm 1;
2. Given a thereshld, "AUD_threshold.m" works out the active devices;
3. "ErrorCalcu.m" calculates the number of error devices, symbols, and bits;
4. "Constell_soft_LLR.m" calculates the LLRs of constellation symbols;
5. "LLR_soft_MBM1203.m" calculates the LLRs of media modulation symbols;
